# QR Code Based Student Attendance Management System

A web application that uses QR codes to manage student attendance efficiently.

## Features
- QR Code Generation for lecturers
- QR Code Scanning for students
- Real-time attendance tracking
- Role-based authentication (Student/Lecturer)
- Attendance history and reports

## Technologies Used
- PHP 8
- MySQL
- Bootstrap 5
- HTML/CSS/JavaScript

## Setup Instructions
1. Clone the repository
2. Import the database from `/sql` folder
3. Configure `config/database.php`
4. Run on XAMPP/WAMP
